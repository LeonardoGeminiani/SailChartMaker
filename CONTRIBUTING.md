# Contributing to SailChartMaker

Thank you for your interest in contributing to SailChartMaker.

## Contributor License Agreement

Before your contribution can be accepted, you must agree to the
[Contributor License Agreement](CLA.md). By opening a pull request or
submitting any contribution, you indicate your acceptance of those terms.

## How to contribute

1. Fork the repository
2. Create a branch for your change
3. Make your changes
4. Open a pull request with a clear description of what you changed and why

## License maintenance

When releasing a new major version, the following fields in `LICENSE` must
be updated:

- `Licensed Work` — update the version number (e.g. `SailChartMaker v2.0`)
- `Change Date` — set to the release date + 4 years (e.g. `2032-03-20`)

This ensures each new version has the full BSL 1.1 protection window.
